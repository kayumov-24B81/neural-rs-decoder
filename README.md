# Neural Network-Assisted Decoding of Reed-Solomon RS(255,223)

[![CI](https://github.com/kayumov-24B81/neural-rs-decoder/actions/workflows/ci.yml/badge.svg)](https://github.com/kayumov-24B81/neural-rs-decoder/actions/workflows/ci.yml)

A neural network predicts error positions in RS(255,223) codewords and passes them
as erasures to a classical decoder, extending correction capability from 16 to up to 32
symbol errors. This is the standard CCSDS code used in space communications (NASA/ESA).

## How it works

```
Message ──► RS(255,223) Encoder ──► Channel (AWGN/QSC) ──► Received word
                                                                │
                                           Syndrome ◄───────────┘
                                              │
                                    ┌─────────▼──────────┐
                                    │   Neural Network   │
                                    │  (error positions) │
                                    └─────────┬──────────┘
                                              │
                                      Erasure mask
                                              │
                                    ┌─────────▼──────────┐
                                    │  Erasure Decoder   │
                                    │   (up to 2t = 32)  │
                                    └─────────┬──────────┘
                                              │
                                     Decoded message
```

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/neural-rs-decoder.git
cd neural-rs-decoder
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

Requires Python ≥ 3.10 and PyTorch ≥ 2.5.

## Quick start

```bash
# Run tests
pytest tests/ -v

# Launch experiments
jupyter notebook notebooks/
```

Run notebooks in order: `00_reproducibility` → `01_awgn_baseline` → ... → `05_soft_input`.

## Repository structure

```
├── src/                 Source code
│   ├── model.py         MLP architecture
│   ├── codec.py         RS(255,223) encode/decode
│   ├── channel.py       QSC and AWGN+BPSK channels
│   ├── dataset.py       Data generation
│   ├── train.py         Training loop
│   ├── evaluate.py      BER/FER metrics
│   └── utils.py         Reproducibility utilities
├── configs/             Experiment configurations (YAML)
├── notebooks/           Jupyter notebooks for experiments
├── results/             Tables (CSV) and figures (PNG)
└── tests/               Unit tests
```

## Experiments

| # | Experiment | Description |
|---|-----------|-------------|
| 0 | Reproducibility | 5 seeds, baseline metrics mean ± σ |
| 1 | AWGN + BPSK | BER/FER curves vs Eb/N0, coding gain |
| 2 | NASA PDS data | Validation on real telemetry messages |
| 3 | Ablation study | Network size, input format, loss function |
| 4 | Robustness | Training on mixture of error distributions |
| 5 | Soft input (LLR) | Log-likelihood ratios as additional input |
